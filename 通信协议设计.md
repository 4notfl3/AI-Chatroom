# 通信协议设计

# 使用websocket进行前后端通信



## 上行：前端 → 后端

### 进入聊天室

```JSON
{
  "type": "join",
  "username": "张三"
}
```

### 退出聊天室

```JSON
{
  "type": "leave"
}
```

### 发送广播消息

```JSON
{
  "type": "chat",
  "target": "broadcast",
  "content": "大家好"
}
```

### 发送私聊消息

```JSON
{
  "type": "chat",
  "target": "user",
  "to": "user_002",
  "content": "你好"
}
```

### 跟大模型聊天

```JSON
{
  "type": "chat",
  "target": "llm",
  "model": "qwen3.6-flash",
  "content": "请介绍一下 FastAPI"
}
```

---

## 下行：后端 → 前端

### 进入成功

```JSON
{
  "type": "join.ok",
  "user_id": "user_001",
  "username": "张三",
  "time": "2026-06-09T10:30:00Z"
}
```

### 有用户进入，广播系统消息

```JSON
{
  "type": "system",
  "event": "user.joined",
  "content": "张三进入了聊天室",
  "user": {
    "id": "user_001",
    "username": "张三"
  },
  "time": "2026-06-09T10:30:00Z"
}
```

### 有用户退出，广播系统消息

```JSON
{
  "type": "system",
  "event": "user.left",
  "content": "张三离开了聊天室",
  "user": {
    "id": "user_001",
    "username": "张三"
  },
  "time": "2026-06-09T10:30:00Z"
}
```

### 收到广播消息

```JSON
{
  "type": "message",
  "target": "broadcast",
  "from": {
    "id": "user_001",
    "username": "张三"
  },
  "content": "大家好",
  "time": "2026-06-09T10:30:00Z"
}
```

### 收到私聊消息

```JSON
{
  "type": "message",
  "target": "user",
  "from": {
    "id": "user_001",
    "username": "张三"
  },
  "to": {
    "id": "user_002",
    "username": "李四"
  },
  "content": "你好",
  "time": "2026-06-09T10:30:00Z"
}
```

### 大模型普通回复

```JSON
{
  "type": "message",
  "target": "llm",
  "from": {
    "id": "llm",
    "username": "qwen3.6-flash"
  },
  "model": "qwen3.6-flash",
  "content": "FastAPI 是一个用于构建 API 的 Python Web 框架。",
  "time": "2026-06-09T10:30:00Z"
}
```

### 大模型流式回复

```JSON
{
  "type": "llm.delta",
  "model": "qwen3.6-flash",
  "content": "FastAPI 是",
  "done": false
}
```

```JSON
{
  "type": "llm.delta",
  "model": "qwen3.6-flash",
  "content": "一个现代 Python Web 框架",
  "done": false
}
```

```JSON
{
  "type": "llm.done",
  "model": "qwen3.6-flash",
  "done": true
}
```

### 错误消息

```JSON
{
  "type": "error",
  "code": "USER_NOT_FOUND",
  "message": "目标用户不存在",
  "time": "2026-06-09T10:30:00Z"
}
```

---

### 用户列表

```JSON
{
  "type": "users",
  "users": [
    {
      "id": "user_001",
      "username": "张三"
    },
    {
      "id": "user_002",
      "username": "李四"
    }
  ],
  "time": "2026-06-09T10:30:00Z"
}
```

```Shell
时机：
用户进入后发送
用户退出后发送
也可以在用户 join.ok 后立即发送
```
